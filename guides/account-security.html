Аккаунт: базовая гигиена

Это не «паранойя» и не «для профи». Это минимальный набор, без которого в крипте долго не живут.

2FA Антифишинг Почта Устройства
База Доступы Ошибки
Коротко: если аккаунт угнали — спорить не с кем. Биржа почти всегда исходит из того, что доступ был у владельца.

Обязательная база

Если здесь есть «нет» — это не чек-лист, а список проблем.

1

2FA включён

Не SMS. Приложение-аутентификатор + сохранённые резервные коды офлайн.

2

Почта отдельная

Почта не используется нигде кроме крипты. Пароль уникальный, 2FA включён.

3

Антифишинг-код

Любое письмо от биржи без кода — в мусор. Без обсуждений.

4

Пароли не повторяются

Биржа ≠ почта ≠ другие сервисы. Менеджер паролей — не роскошь.

Контроль доступов

Большинство взломов — это не «хак», а утекший доступ.

Что включить

  • Белые адреса для вывода
  • Подтверждение вывода через 2FA
  • Уведомления о входе
  • Проверка истории логинов

Что ограничить

  • API без нужды
  • Доступы с чужих устройств
  • Браузерные расширения
  • Общий доступ к ПК
Правило: если вы не понимаете, зачем вам API — значит он вам не нужен.

Типовые ошибки

Их совершают даже опытные — один раз, а потом надолго.

Критичные

  • SMS вместо 2FA
  • Почта без 2FA
  • Переходы по ссылкам из писем
  • Хранение кодов в заметках

Медленные, но опасные

  • Один пароль «на всё»
  • Расширения «для удобства»
  • Общий компьютер
  • Отсутствие проверок логинов

Быстрый чек

Если ответ «да» на всё — вы в зелёной зоне.

Есть

  • 2FA через приложение
  • Резервные коды офлайн
  • Антифишинг-код
  • Белые адреса

Нет

  • SMS-2FA
  • Общих паролей
  • Лишних API
  • Левых расширений
