# USDT Guide (RU)

Небольшой статический сайт-навигатор: **как купить и использовать USDT без лишних рисков и блокировок**.  
Фокус — **новички и “обычные задачи”** (купить/хранить/использовать), без трейдинга и без обещаний дохода.

## Что это
- Пошаговый маршрут (от “что такое USDT” до первого безопасного действия)
- Практическая инструкция покупки USDT
- Страница с типовыми рисками и ошибками
- FAQ в формате аккордеона

## Чего здесь нет
- “Сигналов”, “стратегий”, “гарантированной доходности”
- Управления чужими средствами
- Серых схем и инструкций по обходам

## Для кого
- Новички, которые хотят понять процесс и не накосячить
- Предприниматели и фрилансеры, кому нужен USDT для практических задач

## Стек
- HTML / CSS / минимальный JS
- GitHub Pages (статический хостинг)

## Структура проекта
- `index.html` — главная
- `route.html` — маршрут (карта процесса)
- `why-bybit.html` — почему выбран этот инструмент
- `how-to-buy.html` — пошаговая инструкция
- `risks.html` — риски и типовые ошибки
- `faq.html` — FAQ
- `bybit.html` — страница-редирект под партнёрскую ссылку (удобно менять в одном месте)
- `assets/css/style.css` — стили
- `assets/js/main.js` — автогод и мелкая логика

## Как запустить локально
Самый простой способ — открыть `index.html` в браузере.  
Для удобной разработки можно использовать **Live Server** в VS Code.

## Деплой
